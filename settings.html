Features
  • Clean Taobao Links: Removes unnecessary parameters from Taobao and Tmall links, keeping only the product ID.
  • Clean Shopee Links: Simplifies Shopee links from various countries, keeping only essential parameters.
  • Clean Yahoo Shopping Links: Simplifies Yahoo Shopping links by keeping only the essential product ID and removing all parameters.
  • Clean Rakuten Taiwan Links: Removes tracking parameters (scid, srsltid) from Rakuten Taiwan shopping links.
  • Clean Rakuten Japan Links: Removes tracking parameters (iasid) from Rakuten Japan shopping links.
  • Clean PcHome Links: Removes tracking parameters (srsltid) from PcHome shopping links.
  • Clean Tracking Codes: Removes tracking parameters like UTM, Facebook IDs, and other common tracking codes.
  • Clean All Parameters: Removes all URL parameters, keeping only the base URL.
  • QR Code: Automatically generates a QR code for the cleaned link.
